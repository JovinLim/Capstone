Running the plugin
1. Copy the MyFirstPlugIn.ADDIN file into the folder
C:\ProgramData\Autodesk\Revit\Addins\2022


2. Within the .cs file, change the python directory to the python.exe path

3. Change the path of the referenced python file to the path of the python file called "test2.py" included in the Python folder of the downloaded zip.

Run the plugin through the Add-Ins tab
